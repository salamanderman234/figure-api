Documentation

Endpoint

/api/figures

Description : Get all figures by keyword
Query Params :

keyword

Type : string
Status : required
Description : keyword of an item ex: kato+megumi
Requirements : the more specific the keywords entered, the better the search speed will be
Default : none (can't be empty)
Possible value : text
Example :
/api/figures?keyword=
*edit input value
Result :
{result : asiap}

source

Type : string
Status : optional
Description : source of data
Requirements : must match all possible
Default : hobby-search
Possible value : hobby-search
Example :
/api/figures?keyword=kato+megumi&source=
*edit input value
Result :
{result : asiap}

page

Type : number
Status : optional
Description : pagination
Requirements : if the page filled exceeds the number of pages from the keyword searched, it will return 404
Default : 1
Possible value : any number
Example :
/api/figures?keyword=kato+megumi&page=
*edit input value
Result :
{result : asiap}

item_series

Type : string
Status : optional
Description : series of an item
Requirements : series of an item ex: nendoroid
Default : none
Possible value : text
Example :
/api/figures?keyword=kato+megumi&item_series=
*edit input value
Result :
{result : asiap}

status

Type : string
Status : optional
Description : status of an item
Requirements : must match all possible
Default : none
Possible value : preorder or ready
Example :
/api/figures?keyword=kato+megumi&status=
*edit input value
Result :
{result : asiap}

manufacturer

Type : string
Status : optional
Description : manufacturer of an item
Requirements : manufacturer of an item ex: max+factory
Default : none
Possible value : text
Example :
/api/figures?keyword=kato+megumi&manufacturer=
*edit input value
Result :
{result : asiap}
